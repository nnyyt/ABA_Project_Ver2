High-level explanation of the visible graph for non-technical users

Graph Definition
  1. Top Node - Main Opinion: Read the gray top node first (good_claim or bad_claim). This represents the Main Opinion of the selected category and the final conclusion being evaluated in the graph.
  2. Middle Layer - Supporting Reasons / Opposing Views: The middle layer contains the key reasons connected to the Main Opinion. Green nodes represent supporting reasons, while red nodes represent opposing views that challenge the reasoning path through attack lines. A dashed border indicates an assumption that is not directly evidenced at that layer but is still part of the reasoning structure.
  3. Bottom Layer - Review Evidence: The bottom layer usually contains real customer review phrases. These phrases provide the evidence that supports higher-level reasons or, in some cases, act as opposing assumptions that challenge them.
  4. Arrows and Colors: Follow the arrow direction to understand the flow of influence. Black lines indicate support or connection, while red lines indicate attack or counterargument.
  5. Badge Number: The number shown on a node represents the amount of supporting evidence. A higher number means that point is mentioned in more reviews.
